## 💡 工具洞察

### CLI 转化工具生态总结

1. **Browser → CLI** 正在成为 AI Agent 的标配能力。`agent-browser` 和 `bb-browser` 代表了两种路线：前者偏 CLI 原生，后者通过 MCP 协议标准化
2. **Website → CLI** 目前成熟度较低，`opencli` 是该领域的创新尝试，将任意网站转化为 CLI
3. **NL → SQL** 赛道最成熟，`vanna` 和 `WrenAI` 都有大量实际应用场景
4. **MCP 生态** 正在快速扩张，从 VS Code 扩展到浏览器、设备控制、截图等场景
